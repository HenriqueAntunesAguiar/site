Conceitos de Título e Parágrafo

  • H1

  • H3

Para definirmos o parágrafo temos o p e /p que forma sua estrutura.

Estrutura

Dentro do código temo o head e /head que forma a configuração do site, style, tiltulo, UTF-8 (caracter portugues) antre outros metas.Style é usado para configurar a parte visual com CSS do site dentro da configuração head como visto anteriormente.Para cabeçarios iguais o "HTML5 e CSS" colocamos dentro do body e /body.

Algumas perfumarias

Para utilização de marca texto usamos o mark e /mark
i e /i para itálico u e /u para sublinhado
em e /em para dar ênfase em uma palavra
sup e /sup para números elevados como 512
sub e /sub para números químicos como H2O
strong e /strong para negrito

Listas

Para listas definimos como ul e /ul para listas não ordenadas como as anteriores; ol e /ol usada para listas ordenadas como:

  1. Acordar
  2. Levantar
  3. Escovar os dentes
  4. Colocar a roupa

Lembrando que colocamos li e /li dentro da ol ou ul para definirmos aquilo como pertecente à uma lista. Também podemos definir tipos dentro da lista ao usar li type na ol:

  1. 'a' indica letras minúsculas,
  2. 'A' indica letras maiúsculas,
  3. 'i' indica algarismos romanos minúsculos,
  4. 'I' indica algarismos romanos maiúsculos,
  5. '1' indica números (padrão)

Ancoragem

Para ancoragem usamos a e /a onde a sintaxe é: 

            a href = Link/arquivo html (entre aspas) >Palavra< /a

Para ancoragem de links externo onde queremos abrir em uma janela em branco usamos:

BackGround Image

Vimos que podemos utilizar o background-color no style/body para colocarmos a cor de fundo.
Contudo podemos utilizar a seguinte sintaxe: 

            
        style   
            body
                background-image: linear-gradient(direção, cor1, cor2, cor3, etc)
            /body
        /style

Assim criamos um degrade, podendo colocar como direção em graus com "90deg" ou as quatro direções:

  • to top
  • to right
  • to left
  • to botom

Contudo também temos que escrever no caso de top e botom a seguinte sintaxe: 


        * {
            height: 100%
        }

Onde o * serve para determinar uma configuração global e depois a altura.
Podemos colocar depois da cor a % dela correspondente a tela, assim modificando o degrade.

Textos

Algumas tags de texto:

  • font-family: 'fonte'; - usado para definir a fonte do site
    • Voce pode usar fontes repetidas para caso uma não seja compatível ele usar a príxmo, exemplo:
    • font-family: 'fonte1','fonte2', 'fonte3'
  • font-size: 10em; - usado para definir tamanho da fonte
  • font-weight ; - usado para definir o peso da fonte. Podendo ser light, normal, bold e bolder.
  • font-style ; - usado para deixar o texto em itálico, negrito e etc.

Podemos deixar todas as configurações em uma unica linha usando o font, contudo a ordem dos argumentos tem que ser font:
font-style, font-weight, font-size, font-family
Exemplo:
font: italic bolder 3m 'Arial';
É possível usar a formula com menos um valor, contudo tem que permanecer na ordem.

Também podemos importar fonts usando o Google Fonts em que precisa estar conectado a internet,
contudo não é preciso baixar.
Caso você queira usar uma fonte externa e baixada, deve ser usado o comando font-face ;

Seletores

Para criar uma identificação única do seu elemento no site, devesse usar id='apelida', como na sintaxe: 

        h1 id="principal" /h1

E quando precisar chamar esse id dentro do style usaremos h1#principal{}.

Posso usar a mesma configuração para mais de um elemento, contudo não devo usar id e class
onde sua identificação é o '.'.
Contudo é indicado colocar somente .apelido dentro do style pois podemos chamar posteriormente as configurações da class
em uma palavra usando span class='apelido'> texto /span>.
Podemos utilizar mais de uma classe dentro de class usando somente apelido1 apelido2

Pseudo-Classes

As Pseudo-Classes são definidas com :, diferentemente da class que é . e o id representado por #.
Dentro das Pseudo-Classes temos o :hover que define que quando passamos o mouse por cima de um objeto ele faz uma determinada ação como por exemplo trocar o plano de fundo do texto.
Outra Pseudo-Classe é a :vistied que é usada para definir quando seu link ja foi visitado, por exemplo. 

                a:visited {
                    color: blue;
                    text-decoration: none
                }


Assim quando a pessoa ja entrou ele vai continuar azul porém sem underline. Agora, caso você queira usar o underline quando passar o mouse por cima, basta colocar o :hover e o :active para quando eu clicar. 

                a:hover {
                    color: black;
                    text-decoration: underliner;
                }


Para indicar quem queremos modificar um elemento dentro do .especial do h1, usamos a tag > que significa que aquele elemento é filho de alguma tag como div e etc.

Caixas!

Dentro das caixas temos algumas medidas, sendo elas:

  • hight - altura
  • width - largura
  • border - borda
  • padding - diferença da borda e largura/altura
  • margin - margem
  • outline - contorno/traçado

Tipos de caixa:
box-level: Sempre pula para a linha de baixo do texto e ocupa a largura toda do navegador. (div, h1-h6, p, main, header, nav, article, aside, footer, form, video) inline-level: Desenha do lado da linha e faz o tamanho da caixa exatamente do conteudo dele. (span, a, code, small, strong, em, sup/sub, label, button, input, select)

Grouping Tags

As Grouping Tags são utilizadas para agrupar blocos de código, onde é possível colocar tags dentro delas, como por exemplo main e /main.
Contudo temos algumas tags para listar.

  • nav - utilizada para menus
  • header - diferente de head é para montar o cabeçario.
  • footer - rodapé da página
  • selection - usado para agrupar em geral

Shadow Box

O style box-shadow é bem simples que parte de inserir sombreamento a caixas como essa que está no título, menu ou header do site.
Para utilizarmos ele bastar colocar as quatro medidas em pixel e depois sua cor: 


        box-shadow{
            1px 1px 1px 1px black;
                    }

Arredondamento de Vértices

Para arredondar a borda usamos o código border-radius e o tanto que devemos arredondar em px.
Contudo temos como arredondar separadamente cada borda usando border-bottom-right/left ou border-top-right/left para criar o efeito desejado.

:root

O root é a definição de que todos os outros elementos vão ter a mesma configuração dele, como as variáveis.

Variáveis

As variáveis são tags que trocam seu valor conforme definido. Normalmente definidas dentro do root, as variáveis tem a sintaxe de: 

        --variável: valor;

Podendo ser essa valor uma cor, número ou texto. Para chamar a variável devemos usa a sintaxe: 

        var(--variável)

Reponsividade

Podemos usar as tags min-width e max-width em px para deixar o contudo de texto sem quebrar para uma barra lateral, contudo se passar o min ele acaba quebrando. Para min temos como referência 320px (celulares antigos) e máx 800px.

Agora, para imagens temos como criar o css img { width: 100%} que a imagem vai reduzindo conforme a tela diminui.
Contudo temos a possibilidade de criar a seguinte sintaxe: 

        picture
        source 
        media="max ou min-width:'X'px" (recomendado o max) srcset="caminho da imagem"
        /source
        /picture

Para fazer a imagem caber corretamente no fundo do site temos duas tags, a background-size: cover e a background-size:contain. A cover faz com que a imagem não seja exibida por inteiro, contudo ele se adápta para cobrir o site todo, enquanto o contain exibe tudo mas distorcendo a imagem.
